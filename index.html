lab 3: Three.js KizunaAI

心得: 今天這個作業的內容是將three.js應用於MMD的網頁呈現，因為這次的學內容是我負責的，所以我對於這次的實作部分其實也有些熟悉，但是再demo時候發生一些問題，導致模型有小部分的瑕疵，在之後引入three.js的光源後，模型的呈現有了小幅度的改善，不過後來發現主要的部份是因為 MMDLader再讀取MMD的時候會預設將每一個材質都做發光處理，導致有部分的材質會因為發光而看不到，把發光的部份解決後，基本上就OK了，再將某些材質上色，可以看得更清楚一點。 然後這次應該是算我做的最久的一個 project 了，加了一些東西，也做了一些研究，希望這次的經驗可以在日後派上用場吧
左邊是原本做的版本，傳到 Youtube 上沒有被封鎖，右邊不管怎麼傳都會被封鎖，即使我調過 Midi 音樂還是一樣，最後就直接把原檔放 Github 了，反正才 18mb 左右。
